# Overblik over tags til brug i tidsregistrering

Følgende giver et hurtigt overblik over de tags som anvendes og til hvilken type opgaver (formål).

## Overblik over tags og formål

| Tag                                   | Formål (kort fortalt) |
|----------------------------------------|------------------------|
| Strategi & interessevaretagelse       | Retning, positionering og ekstern strategisk påvirkning |
| Governance & bestyrelsesbetjening     | Formelle beslutningsprocesser og styring |
| Produkt-/projektstøtte (udvikling)    | Fremdrift og udvikling af produkter og projekter |
| Produkt-/projektstøtte (drift)        | Stabilisering og løbende opfølgning på produkter |
| Kommunikation & synlighed             | Formidling, synlighed og repræsentation |
| Fællesskab & relationer               | Medlemsdialog, engagement og samarbejdskultur |
| Administration & intern drift         | Praktisk og administrativ understøttelse |
| Ledelse & personale                   | Ledelse, trivsel og intern prioritering |
| Kompetenceudvikling & læring          | Faglig opdatering og kompetenceopbygning |
| Pause                                 | Betalt pause i arbejdstiden |
| Fravær                                | Ferie, sygdom og andet fravær |
| Møde – internt                        | Internt møde uden tydelig primær fagkategori |
| Transport                             | Transport uden aktivt arbejde |
